# Vuetify Ecosystem Feedback

Central hub for collecting feedback on private Vuetify ecosystem projects.

## Scope

This repository handles feedback for:

| Project | URL |
|---------|-----|
| Play | [play.vuetifyjs.com](https://play.vuetifyjs.com) |
| Bin | [bin.vuetifyjs.com](https://bin.vuetifyjs.com) |
| Studio | [studio.vuetifyjs.com](https://studio.vuetifyjs.com) |
| Snips | [snips.vuetifyjs.com](https://snips.vuetifyjs.com) |
| Link | [link.vuetifyjs.com](https://link.vuetifyjs.com) |
| Issues | [issues.vuetifyjs.com](https://issues.vuetifyjs.com) |

## Submitting Feedback

Use the appropriate issue template:

- **Bug Report** - Something isn't working as expected
- **Feature Request** - Suggest a new feature or enhancement
- **Documentation** - Report docs issues or suggest improvements
- **General Feedback** - Everything else

## Not Covered Here

- **Vuetify framework** - [vuetifyjs/vuetify](https://github.com/vuetifyjs/vuetify/issues)
- **Vuetify 0** - [vuetifyjs/0](https://github.com/vuetifyjs/0/issues)
- **Vuetify One** - [one.vuetifyjs.com/support](https://one.vuetifyjs.com)
